红黑树： 是一个二叉查找树， 在平衡二叉树的高效查找速率和低速插入删除速率的折中。

特性：  
1、每个节点必须是红色或者黑色  
2、根节点必须是黑色  
3、每个叶子节点(null)是黑色  
4、每个红色节点的子节点必是黑色  
5、从一个节点到该节点的子孙叶子节点的所有路径上包含相同数目的黑色节点
 
参考HashMap 中红黑树的实现

上面两步保证了节点的 “特性5”， 但是可能会破坏 “特性2” 或 “特性4”.
